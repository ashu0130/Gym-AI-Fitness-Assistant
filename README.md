# 🏋️ Gym-AI-Fitness-Assistant

This project predicts calories burned during gym workouts using Machine Learning and provides workout recommendations using NLP.

---

## 🚀 Features
- Predict calories burned using ML models
- NLP-based workout recommendation system
- Interactive web app using Gradio
- Deployed on Hugging Face Spaces

---

## 📊 Models Used
- Linear Regression
- Decision Tree
- Random Forest (Best Model)

---

## 🧠 NLP Feature
Users can enter fitness goals like:
- "I want fat loss"
- "Muscle gain"

System suggests workouts accordingly.
